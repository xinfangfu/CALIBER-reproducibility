# Third-party data and code boundary

The project uses multiple public research datasets and external model implementations.  Their
citations and scientific roles are recorded in the manuscript.  This package deliberately
excludes raw trajectories, provider archives, external checkpoints, and copied third-party
source trees.  Cell IDs, split assignments, checksums, and derived aggregate metrics are
included for auditability; inclusion does not assert ownership or relicense the source data.

Before public deposit, the author must verify each provider's current access, attribution,
and redistribution conditions.  If a provider forbids redistribution, the archive should
retain only the manifest/hash and direct readers to the cited source.
